# spring-demo
用Spring+Thymeleaf+MySQL实现了一个简单的可以登录注册的页面，项目来自廖雪峰的java教程，我的代码里做了一些修改和替换。
## 最终效果

### 体验

http://47.98.252.1:8082/

### index页面

如果未登录，index页面会有Sign In按钮，不会有提示消息。

如果已登录，index页面会有Log Out按钮，有welcome+用户名的提示消息。
